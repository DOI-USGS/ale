Orientations inverse() const¶

Invert the orientations. Note that inverting a set of orientations twice does not result in the original orientations. the constant rotation is applied after the time dependent rotation. This means in the inverse, the constant rotation is applied first and then the time dependent rotation. So, the inverted set of orientations is entirely time dependent. Then, the original constant rotations cannot be recovered when inverting again. The set of orientations will still operate the same way, but its internal state will not be the same.
