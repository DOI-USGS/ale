ale.drivers.load(label, props={}, formatter='ale', verbose=False)[source]¶

Attempt to load a given label from all possible drivers

Parameters
labelstr

String path to the given label file

propsdict

A dictionary of optional keywords/parameters for use in driver loading

formatterstr

Formatted output to expect from the driver

verbosebool

If True, displays error messages from driver loading
